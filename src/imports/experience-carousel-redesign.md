Improve the Experience carousel layout to feel more premium and cinematic.

The current carousel feels too constrained and "UI kit-like".

Layout improvements:

1. Full-width carousel

The carousel should not be inside a fixed-width container.

The cards track must span the full viewport width.

The side cards should be partially visible and touch the left and right edges of the screen.

This creates a cinematic portfolio feel.

2. Card hierarchy

The center card must be clearly dominant.

Active card:

* scale 1
* full opacity
* centered

Side cards:

* scale around 0.9
* opacity around 0.4–0.6
* slightly shifted horizontally

This creates depth and focus.

3. Navigation redesign

Remove the circular arrow buttons.

Instead use the timeline as the primary navigation.

Timeline design:

A horizontal line with milestones:
2018 — 2020 — 2022 — 2024 — Future

The active point should be highlighted.

Clicking a point should move the carousel to the corresponding card.

4. Timeline animation

When the active card changes:

* the timeline indicator moves smoothly
* the progress line fills toward the active milestone

5. Card layout improvements

Cards should feel more premium:

* larger padding
* softer borders
* subtle shadow
* slightly larger title
* better vertical spacing between sections

6. "Future / You?" card

The final card should be visually different.

Design:

* dashed border or subtle glow
* centered text
* minimal content

Text:

Title:
Future role

Content:
Maybe your company?

Let's build something meaningful together.

7. Motion

Transitions between cards should feel smooth and fluid.

Use horizontal slide + slight scale transitions rather than abrupt snapping.

Goal:

The carousel should feel like a premium portfolio storytelling component, not a generic UI carousel.
