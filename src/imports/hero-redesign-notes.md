Redesign the hero section to feel more minimal, premium and editorial.

The current version feels too centered and visually busy.

Goals:

* cleaner layout
* more whitespace
* calmer background
* typography-driven design

Layout changes:

1. Text alignment

Move the text block significantly to the left.

The hero content should align with a left content column, not centered in the page.

Use something like:

* margin-left around 8–10vw
* max-width around 650–700px

The right side of the screen should mostly remain empty to create strong visual balance.

2. Typography hierarchy

Keep the main text:

"Hi, I'm Blandine Berthod."
"I'm a product manager"

But improve hierarchy:

Line 1 smaller
Line 2 much larger and stronger

Example:

Hi, I'm Blandine Berthod.
I'm a product manager

Then the description below in smaller text.

3. Background simplification

The background grid should be much more subtle.

Reduce:

* opacity of the grid lines
* contrast
* visual noise

The grid should almost disappear unless you look carefully.

4. Color treatment

The current purple/red/blue gradient feels too strong.

Replace it with much softer blurred color areas.

Use large blurred gradients with low opacity.

Example approach:

* one subtle purple glow on the left
* one soft blue glow on the right

Use heavy blur and low opacity so the colors feel atmospheric rather than decorative.

5. Whitespace

Increase vertical spacing between:

headline
description text
scroll indicator

The hero should feel calm and spacious.

6. Overall tone

The hero should feel inspired by:

Linear
Vercel
Stripe
Ashby

Focus on typography and space rather than graphic effects.

Avoid:

* strong gradients
* visible grid lines
* centered layout
* busy background
